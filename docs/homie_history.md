## homie history

List clipboard history

### Synopsis

List clipboard history
  Use <tab> to pin and select multiple entries

```
homie history [flags]
```

### Options

```
  -h, --help        help for history
  -l, --limit int   Limit the number of clipboard history items displayed (default 20)
  -p, --paste       Paste selected history item
```
